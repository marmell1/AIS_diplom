# AIS_diplom

**Таблицы  - models.py**

Products - каталог товаров (в данном кейсе продукты питания)  
Warehouses - каталог складов  
WarehousesLoading - загрузка складов на текущий момент  
StockMovement - реестр движения товаров

**Подготовка к работе**
create_tables() - создание таблиц в БД  
filling_data() - заполнение первоначальными данными  
prepare() - подготовка данных и проверки (не закончено)

все вместе запускается start.py

**Движение по складам**
_4_Processing  
handler - консолидация двух функций:  
change_wh - внести изменения в WarehousesLoading (приход/расход)  
add_stock_movement - внести изменения в реестр

**Ввод** main.py
CLI - move (typer)

 * │
│ *  --q             INTEGER  Количество товара [required]         
 * │
│ *  --source        INTEGER  Склад, с которого списываем либо поставщик [required]
